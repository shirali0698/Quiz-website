ANDROID LEARNING

Android is a mobile operating system based on a modified version of the Linux kernel and other open source software, designed primarily for touchscreen mobile devices such as smartphones and tablets.

SO WHAT IS ANDROID?

Android is a free and open source software where its source code is known as Android Open Source Project (AOSP) which is primarily licensed under the Apache License.Android is developed by a consortium of developers known as the Open Handset Alliance and commercially sponsored by Google. It was unveiled in 2007, with the first commercial Android device launched in September 2008.

ANATOMY OF ANDROID

The anatomy of android is as follows :
  1. Android Studio: This is used to organize the files and code that make up your app, or maybe you’re using another IDE like Eclipse.
  2. AndroidManifest.XML: This file contains a bunch of properties that you’ll need to set when you eventually deploy your app to the Play Store or on other phones. This is where stuff like the name of your app and its permissions gets set.
  3. Source Code: Android apps are written in Java and their source codes are stored in .java files. The .java files are stored in whatever package you chose when you created your app project. The entry point of an Android app is the main activity class.
  4. Resources: The resources directory contains non-code files that are needed to run your app. Stuff like images and property files go here.Some of its sub-directories are drawable,layout,mipmap,values
